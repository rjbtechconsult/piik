**Piik** is a premium, lightning-fast macOS menu bar application designed for developers who live in Azure DevOps. Stay in your flow and manage your tasks, stories, and bugs without ever leaving your active window.

## Key Features

- 🚀 **Instant Access**: Zero-latency access to your work items via the macOS tray.
- 🎨 **Premium UX**: Modern, dark-themed interface with glassmorphism and smooth animations.
- 🔔 **Smart Notifications**: Stay updated with aggregated, non-intrusive alerts.
- 🔗 **Direct Links**: Quickly copy work item URLs or open them in your browser.
- ⚡ **Native Performance**: Built with Tauri and Rust for maximum efficiency.
